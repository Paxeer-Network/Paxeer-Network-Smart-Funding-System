# Examples

This directory contains usage examples for the Paxeer User Stats API SDK.

## Files

- **health.ts** — Examples for Health API endpoints
- **tokens.ts** — Examples for Tokens API endpoints
- **portfolio.ts** — Examples for Portfolio API endpoints
- **pnl.ts** — Examples for PNL API endpoints
- **charts.ts** — Examples for Charts API endpoints
- **quick-start.ts** — Quick start guide with basic setup

## Running Examples

```bash
npx ts-node examples/quick-start.ts
```
